## Project Description

Quakescape is a geospatial risk intelligence platform for emergency managers, infrastructure planners, and disaster researchers. It combines live USGS earthquake feeds with Open-Meteo weather data and OSM Nominatim place intelligence to help users understand not just where an earthquake occurred, but what communities, assets, and environmental conditions may be affected.

The platform supports operational awareness workflows with a live seismic map, event drill-downs, nearby place resolution, weather-complication context, and comparative analysis across regions and time. It is designed to help teams prioritize inspections, plan field response, and monitor cascading hazard conditions in one integrated interface.

## Features

- Live earthquake map with clustered seismic activity
- Event drill-down views with nearby place intelligence
- Weather risk context for heavy rain, wind, temperature swings, and forecasted precipitation
- Comparison views for regional and historical analysis
- Timeline and chart-based analysis for magnitude, depth, and aftershock patterns
- Search and filtering tools for exploring active events
- Operational dashboard layout for emergency response workflows
